Zipkin Server
------
install
```
curl -sSL https://zipkin.io/quickstart.sh | bash -s 
```
start server
```
java -jar zipkin.jar
```
default dashboard link: 
[localhost:9411/zipkin/](http://localhost:9411/zipkin/)
